# Contributing to reqstool-demo

Thank you for your interest in contributing!

For DCO sign-off, commit conventions, and code review process, see the organization-wide [CONTRIBUTING.md](https://github.com/reqstool/.github/blob/main/CONTRIBUTING.md).

## Prerequisites

- Java 21+
- Maven 3.9+ or Gradle 9.3+
- Python 3.8+ (for reqstool client)

## Setup

```bash
git clone https://github.com/reqstool/reqstool-demo.git
cd reqstool-demo
```

## Build & Test

### Maven

```bash
mvn clean verify
```

### Gradle

```bash
gradle clean build
```
